# campusguidence
## 📖 Overview
The Campus Guidance Web Application is designed to help new students easily navigate and understand the college campus. It provides centralized access to important information such as announcements, attendance, department details, campus map, and helpline services.
The application is developed using HTML, CSS, and JavaScript for the frontend, and Node.js with Express.js for the backend. It uses RESTful APIs to enable communication between frontend and backend.
Users can log in using their credentials and access features like announcements, department information, attendance tracking, and campus navigation.

## 🎯 Dashboards:
- User Login (Student & Admin)
- Campus Map Integration
- Announcements Section
- Department & Faculty Information
- Attendance Tracking System
- Helpline & Contact Details



## 💻 Technologies Used:
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Tools: VS Code, GitHub  



## 🎯 Objective:
- To help new students easily navigate the campus  
- To provide all important information in one platform  
- To improve communication through announcements  
- To build a full-stack web application  



## 🚀 Outcome:
The project successfully delivers a functional campus guidance system where users can log in and access various campus-related services. It demonstrates key concepts of full-stack development including frontend design, backend API creation, and API integration.
